### Build from Source
1. **Clone the Repository:**
   ```bash
   git clone https://github.com/Coconut-audio/there-will-not-be-blood.git
   cd there-will-not-be-blood
   ```
3. **Build the Plugin:**
   ```bash
   cmake -B Builds -DCMAKE_BUILD_TYPE=Release
   cmake --build Builds --config Release
   ```
